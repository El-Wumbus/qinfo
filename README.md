# qinfo

A Linux system info program. Fetches system info and displays it.  

## Compile from source (Linux)

```bash
git clone https://github.com/El-Wumbus/qinfo.git
cd qinfo
sudo make install
```

## Compile with pkgbuild (Arch Linux)

```bash
git clone https://github.com/El-Wumbus/qinfo.git
cd qinfo/archpkg
makepkg -si
```
